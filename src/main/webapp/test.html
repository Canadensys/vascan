Name search

Welcome to the Database of Vascular Plants of Canada (VASCAN). Search for any scientific or vernacular name. A search on "arte" will return "Artemisia", but not "Two-parted sedge". Use "Achnella" to search for hybrids like "×Achnella". Want to search by distribution instead? Use the checklist builder.
